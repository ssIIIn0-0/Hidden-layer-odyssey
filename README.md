# Hidden layer odyssey

## 프로젝트 소개
스마일게이트의 로스트아크 인게임을 기반으로 이스터에그를 추가한 2D 횡스크롤 미니게임

## 개요
- 개발목적 : 2023년 4-1학기 게임프로그래밍과목 기말 팀프로젝트
- 개발기간 : 2023.05.15 ~ 2023.06.12 (약 4주)
- 프로젝트 팀원 : 김명학, 이도희, 박시은
- 개발도구 : Unity 2021.1.25f1
- 개발언어 : C#
- 플랫폼 : PC
